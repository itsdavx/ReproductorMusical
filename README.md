# Reproductor Musical 

## Descripción

Reproductor Musical Interactivo es una aplicación desarrollada en C# que combina reproducción de audio con animaciones gráficas sincronizadas en tiempo real. El sistema permite visualizar distintos efectos visuales que reaccionan dinámicamente al comportamiento de la música mientras se reproduce.

## Objetivos

Aplicar conceptos de computación gráfica mediante el desarrollo de animaciones y efectos visuales sincronizados con la reproducción de audio en tiempo real.

## Funcionalidades

- Cargar archivos de audio.
- Reproducir, pausar y detener canciones.
- Controlar el volumen.
- Visualizar el tiempo de reproducción.
- Cambiar entre distintos modos de visualización.
- Mostrar animaciones sincronizadas con la música.
- Renderizado gráfico en tiempo real.

## Técnicas Implementadas

- Transformaciones geométricas.
- Manejo de coordenadas y objetos gráficos.
- Animaciones en tiempo real.
- Renderizado dinámico.
- Manipulación de colores.
- Sistemas de partículas.
- Sincronización entre eventos multimedia y representación gráfica.

## Tecnologías Utilizadas

- C#
- Windows Forms
- GDI+
- Programación Orientada a Objetos
